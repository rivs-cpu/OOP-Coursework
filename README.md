# Real-Time Event Ticketing System

## Purpose
The Real-Time Event Ticketing System is intended to replicate a real-world ticketing platform in which event tickets are constantly added and purchased in real-time. Through the use of contemporary web technologies, this project seeks to give practical experience with concurrency management, multi-threading, and real-time data synchronisation.Â
